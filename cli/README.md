# RohTembak

CLI client for a certain Indonesian mobile internet service provider.

## Setup (Termux)

```bash
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/rohcuan/rohtembak
cd rohtembak
bash setup.sh
```

## Configuration

Create a `.env` file and fill in the environment variables (obtain them from [Telegram](https://t.me/alyxcli)).

## Run

```bash
python main.py
```
